## 🚀 How to Use

### 1. Install Dependencies (if not already done)
```bash
cd services/core
uv sync
```

### 2. Start the Core Service
```bash
cd services/core
uv run uvicorn core.asgi:application --host 0.0.0.0 --port 8003 --reload
```

### 3. Access Documentation
Open your browser to:
- **Swagger UI**: http://localhost:8003/api/docs/
- **ReDoc**: http://localhost:8003/api/redoc/
- **OpenAPI Schema**: http://localhost:8003/api/schema/

### 4. Test Authentication
1. Open Swagger UI
2. Use the Auth service `/api/auth/login/` endpoint to get a token
3. Click the "Authorize" button (🔓)
4. Enter: `Bearer YOUR_ACCESS_TOKEN`
5. Test any Core service endpoint!
